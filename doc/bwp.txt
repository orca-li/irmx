BIT WISE PRINTER(bwp)
--------------------------------------------------------------------------
--- Bit Output Functions -------------------------------------------------

void
bwp_bit_line(
    uint32_t bits,          // bit string
    size_t bitc             // how many bits to output
);
--- Description ---
Outputs bits starting from the highest bit, and finishes output while less
than bitc

void
bwp_bit_separator(
    uint32_t bits,          // bit string
    size_t sctc,            // number of sections
    const uint8_t *scts     // section sizes
);
--- Description ---
Outputs bits separated by spaces, according to `scts` format
